# Rania Atmani Portfolio Website

This is a starter portfolio website built with HTML, CSS, and JavaScript.

## Files

- `index.html` — main website content
- `style.css` — website design and layout
- `script.js` — mobile menu behavior
- `assets/` — folder for images, resume, screenshots, etc.

## How to edit it

Open `index.html` and replace:

- `your.email@example.com`
- `github.com/yourusername`
- `linkedin.com/in/yourusername`
- The `#` links inside each project card
- Any project descriptions you want to improve

## How to add your resume

1. Add your resume PDF into this folder.
2. Rename it exactly:

```text
resume.pdf
```

3. The "View Resume" button will work automatically.

## How to publish on GitHub Pages

1. Create a new GitHub repository named:

```text
yourusername.github.io
```

Example:

```text
raniaatmani22.github.io
```

2. Upload all files from this folder.
3. Go to the repository settings.
4. Open Pages.
5. Set the source to the `main` branch and the root folder.
6. Your site should publish at:

```text
https://yourusername.github.io
```
